# Customer Churn Analysis

### Project Overview

This project focuses on analyzing customer churn data from a telecommunications company. The objective was to understand customer behavior, identify churn patterns, and create visualizations that provide meaningful business insights.

### Tasks Completed

#### Task 1 - Data Overview and Simple Sorting

* Explored dataset structure
* Sorted customers by tenure
* Filtered churned customers

#### Task 2 - Churn Count Visualization

* Created column chart for churned vs non-churned customers
* Added labels and formatting

#### Task 3 - Monthly Charges Distribution

* Created histogram for monthly charges
* Analyzed distribution patterns

#### Task 4 - Churn Rates by Gender

* Built pivot tables
* Created clustered bar chart comparing churn behavior across genders

#### Task 5 - Monthly Charges and Tenure Heatmap

* Analyzed relationship between tenure, monthly charges, and churn behavior
* Created heatmap visualization

### Tools Used

* Microsoft Excel
* Pivot Tables
* Data Visualization
* Data Analysis Techniques

### Key Insights

* Customers with shorter tenure showed higher churn rates.
* Monthly charges had a noticeable impact on customer churn.
* Gender-based churn differences were analyzed using pivot tables and charts.
* Visualizations helped identify important customer retention patterns.
